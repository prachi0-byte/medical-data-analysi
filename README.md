# Medical Data Cleaning & Analysis Project

## 📌 Overview
This project focuses on cleaning and analysing a real-world medical dataset with missing values, duplicates, and inconsistent formats.

## 🚀 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL

## 🔧 Features
- Data Cleaning (missing values, duplicates, formatting)
- Exploratory Data Analysis (EDA)
- Data Visualisation
- MySQL Integration

## 📊 Key Insights
- Age distribution patterns
- Disease frequency analysis
- Gender-based trends
